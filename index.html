mediaQuery.js

  • An extension and plugin for jQuery to help handle media queries in javascript
  • The first component, the extension, will accept strings and query them against the viewport.
  • For example, "$.mediaQuery('>320');" will check if the current viewport is greater than 320px wide
  • Full logic capabilities are available, meaning it will accept queries such as "(>320 && <768) || (>920 && <1200)"
  • The plugin will run up to 2 functions against the jQuery collection upon which it is invoked when the passed media query is matched or unmatched, respectively.
  • For example: "$(el).mediaQuery('>768', matchingFunction, unMatchingFunction);" will run the matching function when the window is above 768px and the unmatching function when the window is 768 or below (with optional toggle so it only runs once!)
  • Please visit on GitHub for more information!
